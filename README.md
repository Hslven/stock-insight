### 专业名词

1. 开盘到收盘的绝对差价
   涨跌额 / 日内价差 - Daily Price Change / Close-Open Spread
   （简写：ΔP 或 OC Spread）

2. 开盘到收盘的百分比变动
   日内收益率 / 涨跌幅 - Daily Return / Daily Percentage Change
   （简写：Ret 或 %Δ，公式：(Close - Open)/Open \* 100%)
